## レイアウト

```
################

#.....#........#

#..##......#..N#

```

## 凡例

- `#`: 木立（通行不可）
- `.`: 地面（通行可）
- `@`: 入口（村から来たときの位置）
- `v`: 村へ戻る移動口
- `N`: NPC配置点

## 配置

- N: npc=forest-dweller

## 接続
